# Science in Motion

Bilingual Greek/English static website for personal training, exercise science, biomechanics and online coaching.

## Structure

```text
scienceinmotion_site/
├─ index.html
├─ styles.css
├─ script.js
├─ README.md
└─ assets/
   ├─ banner/
   └─ services/
```

## How to preview locally

Open `index.html` directly in a browser.

## How to publish with GitHub Pages

1. Create a GitHub repository named `scienceinmotion`.
2. Upload all files and folders from this package to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.
